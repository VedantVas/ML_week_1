📘 Week 1: Python Fundamentals

Explore the basics of Python programming with hands-on examples.

🎯 Learning Goals

  • Understand the Python programming basics.
  • Learn about variables, data types, operators, and control flow.
  • Write simple programs using loops and conditions.
  • Start building problem-solving skills.

🔹 1. Introduction to Python

Python is a high-level, interpreted language that is easy to read, write, and debug. Its wide usage in various fields from web development to data science makes it a powerful tool for any developer.

💡 Real-World Example:

Python is used in Google Search algorithms, Netflix recommendation systems, and NASA’s data analysis tools.

🔹 2. Variables and Data Types

Variables store data values. In Python, you don't need to explicitly declare their type. 

name = "Vedant"
age = 21
height = 5.9
is_student = True

Python has several built-in data types:

  • int → Whole numbers (10)
  • float → Decimal numbers (3.14)
  • str → Text ("Hello")
  • bool → Boolean values (True, False)

💡 Real-World Example:

Storing student information in a university database: name (str), age (int), GPA (float), scholarship status (bool).

🔹 3. Operators

Operators perform operations on variables and values. Python supports a wide range of operators, including arithmetic, comparison, and logical. 

a, b = 10, 3
print(a + b)   # 13
print(a ** b)  # 1000 (10^3)
print(a > b)   # True

💡 Real-World Example:

An e-commerce app calculates the total price of items in a cart (arithmetic) and applies a discount if the total is greater than 500 (comparison).

🔹 4. Control Flow (Decision Making)

Control flow statements execute code based on conditions. The most common is the if-elif-else block. 

marks = 72
if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")

💡 Real-World Example:

An ATM machine checks if your balance is greater than or equal to the withdrawal amount to allow the transaction.

🔹 5. Loops

Loops are used to execute a block of code repeatedly. Python has for and while loops. 

# For Loop
for i in range(5):
    print("Hello", i)

# While Loop
n = 5
while n > 0:
    print(n)
    n -= 1

💡 Real-World Example:

A for loop can send bulk emails to a list of students, while a while loop can repeatedly ask for a password until the correct one is entered.

🔹 6. Input/Output

The input() function gets user input, and print() displays output. 

name = input("Enter your name: ")
print("Welcome,", name)

💡 Real-World Example:

Online forms ask users for details and then store them in a database.

🔹 7. Type Casting

Type casting allows you to convert a variable from one data type to another. 

x = "100"
y = int(x)  # Convert string to integer
print(y + 10)  # Output: 110

💡 Real-World Example:

Data cleaning in Machine Learning often requires converting string numbers like "123" into integers to perform calculations.

🔹 8. Comments & Documentation

Comments are notes that are ignored by the Python interpreter. They are crucial for making your code readable. 

# This is a single-line comment

"""
This is a
multi-line comment
"""

💡 Real-World Example:

Comments help developers working on a team to understand each other’s code and collaborate effectively.

📝 Practice Questions – Week 1 (Python Fundamentals)

🔹 Level 1: Beginner

  • Write a program to take a user’s name and age as input and print a message showing how old they will be after 50 years.
  • Write a program to print the multiplication table of 7 using a loop.
  • Take 3 numbers as input and print the largest number.

🔹 Level 2: Intermediate

  • Write a program to calculate the sum of all numbers from 1 to 100 using a loop.
  • Take marks of 5 subjects as input, calculate the average, and assign a grade:
    • Average ≥ 90 → Grade A
    • 60 ≤ Average < 90 → Grade B
    • Else → Grade C
  • Write a program that asks the user for a number and checks if it is prime or not.

🔹 Level 3: Challenge

  • Write a program that asks the user to guess a number between 1 and 10. If correct, print "You win!". If wrong, keep asking until correct.
  • Write a program to check if a given string is a palindrome (same forwards and backwards).
  • Write a program to count how many vowels (a, e, i, o, u) are present in a given string.

🔹 Mini Real-World Exercises

  • ATM Simulation: Take user balance as input. Ask for withdrawal amount. If balance is sufficient, subtract and print new balance. Else print "Insufficient Balance".
  • Student Pass/Fail Checker: Take marks of a student as input. If marks ≥ 40, print "Pass". Else, print "Fail".
